# ☕ Coffee Sales Dashboard

An interactive Excel dashboard designed to analyze coffee sales performance across customers, countries, coffee types, roast levels, and package sizes. The dashboard enables users to explore sales trends, identify top-performing customers, and gain actionable business insights through dynamic filtering and visualizations.

## 📊 Dashboard Features

* Interactive Date Timeline
* Roast Type Filtering (Dark, Light, Medium)
* Package Size Analysis
* Sales by Country
* Top 10 Customers by Sales
* Coffee Type Sales Trends
* Dynamic Visualizations and KPIs

## 🔍 Key Insights

* The United States generated the highest sales revenue.
* Arabica coffee recorded the strongest sales growth during the selected period.
* A small group of customers contributed significantly to total sales.
* Sales performance varied across roast types and package sizes, highlighting customer preferences.
